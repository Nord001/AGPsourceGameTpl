Content timeline is powerful and lightweight jQuery plugin, best for displaying date organised content. It is fully customizable, and easily integrateable with any js script, video, flesh etc. Well structured code, and wide ranged API functions make it simple to get started and flexible for customization.

How does it work

  1. Take any content
  2. Sort it by date
  3. Call function
  4. You are ready to go!

Calling

All you need to do is call:

$('#myTimeline').timeline();

And format elements like this:

<div id="myTimeline">

  <div class="item" data-id="26/08/2012" data-description="Some description...">
	... Shorter version of content ...		
  </div>
  <div class="item_open" data-id="26/08/2012">
	... Detaild version of content ...		
  </div>
  
  .
  .
  
</div>


* Can be ID or Class of element.
* Default classes, can be changed on function call.
* Date of element (DD/MM/YYYY format).
* Optional, short description.

Every element is generated with necessary CSS properties so all that's left for you to do is change backgrounds, colors and position them to your liking. If you don't want to go overboard with restyling you can always use one of our styles shipped with plugin and tweak them to suit your taste.

Only thing that u can't change are margines and padding of '.item' and 'item_open' elements, this must be done through parameters.

Parameters

You can modify timeline by using optional parameters argument: 
$('#myTimeline').timeline({
  name : value
});

List of available parameters:

Name Default Description
itemClass '.item' Class used for items
itemOpenClass '.item_open' Class used for item details
openTriggerClass '.item' Class of read more element (default uses whole '.item' to trigger open event), if you change it have in mind that element you use it on should have 'data-id' with value same as parrent '.item'.
closeText 'Close' Text of 'close' button in open item (can be left blank)
timelineWidth 920 Width of timeline bar, timeline arrows are absolute and can go out of this range
itemWidth 240 Width of timeline items
itemOpenWidth 490 Width of item details : '.item_open' element.
itemMargin 10 Spacing between items
scrollSpeed 500 Transition speed between two elements.
startItem 'last' Timeline start item 'data-id' ('last' or 'first' can be used insted).
easing 'easeOutSine' JQuery.easing function for animations

Methods

Methods are called through '.timeline' function:
Name Description
$.timeline('init') Initialises timeline
$.timeline('destroy') Clears timeline data
$.timeline('left') Moves one left by one element
$.timeline('right') Moves right by one element
$.timeline('open', id) Opens element with 'data-id' = id
$.timeline('close', id) Closes element with 'data-id' = id
$.timeline('goTo', id) Goes to element width 'data-id' = id

Methods binded on keyboard events:

Event Method
On KeyPress (left) trigger $.timeline('left')
On KeyPress (right) trigger $.timeline('right')

Events

Events are called through '.timeline' class:
Name Description
'init.timeline' Triggered when timeline is initialised
'scrollStart.timeline' Triggered when item move animation starts
'scrollEnd.timeline' Triggered when item move animation ends
'itemOpen.timeline' Triggered on click to open item
'itemClose.timeline' Triggered on click to close item

Requirements

Required: jQuery 1.3+
Extra: Use jQuery Easing for additional easing options.
